# DataStructure

資料結構課程練習與作業整理。

## 環境需求

- Python 3.x
- numpy（執行 W3_Exercises.py 需要）

## 專案目錄

```text
資料結構/
└─ DataStructure/
	 ├─ LoginSystem.py
	 ├─ README.md
	 ├─ W2_Exercises.py
	 └─ W3_Exercises.py
```

## 檔案說明

### DataStructure 資料夾

- LoginSystem.py
  - 與 login.py 內容相同的登入練習程式。
  - 提供 a（註冊）、b（登入）、c（離開）三種操作。

- W2_Exercises.py
  - 第 2 週 Python 基礎練習集合。
  - 內容包含迴圈、條件判斷、巢狀索引、字串與串列操作等題目。
  - 多數程式碼以註解形式保留，作為題目與參考答案。

- W3_Exercises.py
  - 第 3 週 numpy 練習。
  - 內容包含陣列切片、元素乘法、按軸加總與條件篩選等操作。
  - 檔案內含可直接執行的範例程式碼與輸出示範。

- README.md
  - 本說明文件，記錄專案結構與檔案用途。
